Gleichungen lösen

Du formst beide Seiten gleich um, bis die gesuchte Variable allein steht.

Lineare Gleichung

ax + b = c → x = (c - b) / a

x
Gesuchte Zahl.
a
Faktor vor x. Wenn a = 0 ist, ist es keine normale lineare Gleichung mehr.
b
Zahl, die zu ax addiert wird.
c
Wert auf der rechten Seite.

Beispiel

3x + 5 = 20
3x = 15
x = 5

Die Probe zeigt, ob das Ergebnis stimmt: 3 · 5 + 5 = 20.
