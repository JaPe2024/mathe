pq-Formel

Die pq-Formel löst quadratische Gleichungen in Normalform mit Leitkoeffizient 1.

Formel

x² + px + q = 0
x₁,₂ = -p/2 ± √((p/2)² - q)

x
Gesuchte Zahl oder Nullstelle.
p
Zahl vor x, nachdem die Gleichung auf x² + px + q = 0 gebracht wurde.
q
Zahl ohne x in der Normalform.
x₁, x₂
Die beiden möglichen Lösungen. Manchmal gibt es nur eine oder keine reelle Lösung.

Wann darf ich sie benutzen?

Nur wenn vor x² eine 1 steht. Aus 2x² + 8x + 6 = 0 machst du zuerst durch 2 teilen: x² + 4x + 3 = 0. Dann ist p = 4 und q = 3.

Diskriminante: D = (p/2)² - q

Ist D positiv, gibt es zwei Lösungen. Ist D gleich 0, gibt es eine doppelte Lösung. Ist D negativ, gibt es keine reelle Lösung.
